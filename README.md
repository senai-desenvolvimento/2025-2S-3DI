# 💻 Desenvolvimento de Sistemas — Materiais da Turma

Repositório de conteúdo dos instrutores para o **3º semestre** do Técnico em Desenvolvimento de Sistemas — SENAI-SP. Aqui você encontra slides, exercícios, projetos-guia e tudo que vamos usar ao longo do semestre. 📚

> 📅 **Semestre:** 24/07 a 18/12 · **38 aulas** · organizado em 4 sprints
> 🎯 Consulte este repositório sempre que precisar do material de uma aula.

---

## 🧭 Como usar este repositório

Cada sprint tem sua pasta, e dentro dela o conteúdo segue a ordem das aulas. O caminho é simples:

1. 📂 Entre na pasta da **sprint atual**
2. 📄 Abra o material da aula do dia (slides, roteiro ou exercício)
3. 💡 Siga os exemplos e resolva os exercícios propostos
4. 🙋 Traga suas dúvidas para a aula seguinte

> ⭐ **Dica:** dê uma "estrela" no repositório e mantenha ele nos favoritos do navegador pra achar rápido.

## 🛠️ Prepare seu ambiente

Antes da primeira aula de cada sprint, deixe as ferramentas instaladas:

- 🧩 **VS Code** — nosso editor em todas as sprints
- 🟣 **.NET 10 SDK** — para o back-end (Sprint 2)
- 🟢 **Node.js** — para front-end e mobile (Sprints 3 e 4)
- 📮 **Postman** ou **Insomnia** — para testar as APIs
- 🔺 **Git** — para baixar o material atualizado

> 💾 **Para baixar o conteúdo:** use o botão verde **Code → Download ZIP**, ou clone com `git clone` se já souber usar o Git.
